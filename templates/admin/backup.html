Backup erstellen

Vollständiges Backup

Erstellen Sie ein vollständiges Backup der Anwendung. Das Backup enthält:

  • Datenbank: Vollständiger SQL-Dump aller Tabellen und Daten
  • Dokumente: Alle hochgeladenen Dateien
    • Profilbilder
    • Zertifikate
    • Lebensläufe (CV-Dateien)
Hinweis: Das Backup wird als ZIP-Datei heruntergeladen. Der Download startet automatisch nach dem Klick auf "Backup erstellen".
Wiederherstellung

Um ein Backup wiederherzustellen:

  1. Entpacken Sie die ZIP-Datei
  2. Datenbank: Führen Sie die Datei database.sql in Ihrer MariaDB/MySQL-Datenbank aus: 
    mysql -u benutzername -p datenbankname < database.sql
  3. Dateien: Kopieren Sie den Inhalt des uploads/ Verzeichnisses in das entsprechende Verzeichnis der Anwendung
